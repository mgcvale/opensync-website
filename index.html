What is opensync?

A self-hosted and lightweight image and file storage platform, with all the code and documentation you need to set up your own cloud storageserver.

Lightweight

Opnsync is built with performance and speed in mind, using SQLite and C for minimal overhead.

Open Source

It is completely open source, licensed under GPL 3.0, with no telemetry or financial motivations.

Minimal

Opensync is built to be simple and minimal, with only the necessary features.
